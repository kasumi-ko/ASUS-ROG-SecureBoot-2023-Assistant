# Security Policy

## Do not upload publicly

- Raw Default Keys backups
- BitLocker recovery keys
- Personal files
- Recovery files that contain device backups
- Files that include device serial numbers or full local log paths

For Issues, use the diagnostic report when possible. Check the file yourself before uploading it.

## Notes

The app reads Secure Boot related state. It performs writes only after user confirmation.

The app does not upload logs or backups automatically.
